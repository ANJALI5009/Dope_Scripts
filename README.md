## Commands Used
### 1) To clone Repo
`git clone Repo_URL`

### 2) To create a branch
`git checkout -b Branch_name`

### 3) To Add changes
`git add .`

### 4) To commit changes
`git commit -m 'Your message'`

### 5) To push Changes
`git push GIT_URL Branch_name`
